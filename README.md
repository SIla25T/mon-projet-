## Projet 
Combinaison de deux ontologies de narration

## Description
Prendre deux ontologies (fantastique) et les fusionner en une ontologie enrichie. Le programme détecte les éléments communs, résout les conflits (personnages homonymes, lieux similaires) et produit une ontologie combinée avec un rapport de fusion.
